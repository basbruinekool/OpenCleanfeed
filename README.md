# OpenCleanfeed
DreamHack Open Cleanfeed automation
Requirements 
- JustMacros and PRO license - mail bas.bruinekool@dremahack.se for DreamHack License activation
- ATEM Software control panel
- Active ATEM device

Place the Lua scripts in your JustMacros installation directory, inside the Macros folder
Start JustMacros
Execute the check10s file
To test or monitor changes, open the ATEM Software control panel. Note: this is not required to be open as JustMacros communicates over UDP.

Troubleshooting:
Check if JustMacros can see the correct device in the DEVICES box.
If it cannot, make sure you have the ATEM Software control panel installed and connect to the correct mixer.
